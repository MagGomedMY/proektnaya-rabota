## Основные результаты курсового проекта

### Цель работы
Разработка прототипа веб-интерфейса для интернет-магазина книг с использованием HTML, CSS и JavaScript.

### Ключевые задачи
- Анализ аналогов и обоснование выбора технологий
- Проектирование навигации и пользовательских сценариев
- Реализация базового функционала (каталог, корзина, оформление заказа)
- Тестирование (модульное, интеграционное, юзабилити)
- Оценка эффективности (время на операции, удовлетворённость)

### Выбранный стек

| Технология | Роль | Обоснование |
|------------|------|-------------|
| **HTML** | Структура страниц | Семантическая разметка, доступность |
| **CSS** | Адаптивный дизайн (Material Design) | Flexbox/Grid, медиа-запросы, плавные анимации |
| **JavaScript** | Интерактивность, работа с данными | ES6+, события, манипуляции DOM |
| **localStorage** | Хранение данных корзины | Сохранение между сессиями без сервера |

## Архитектура прототипа

### Структура приложения
- **Одностраничное приложение (SPA)** в файле `index.html`
- **Разделение ответственности:**
  - `index.html` - разметка и структура
  - `css/style.css` - стилизация и адаптивность
  - `js/main.js` - бизнес-логика и интерактивность
