# Книжный магазин - Курсовой проект

## Основные результаты курсового проекта

### Цель работы
Разработка прототипа веб-интерфейса для интернет-магазина книг с использованием HTML, CSS и JavaScript.

### Ключевые задачи
- Анализ аналогов и обоснование выбора технологий.
- Проектирование навигации и пользовательских сценариев.
- Реализация базового функционала (каталог, корзина, оформление заказа).
- Тестирование (модульное, интеграционное, юзабилити).
- Оценка эффективности (время на операции, удовлетворённость).

### Выбранный стек

| Технология | Роль |
|------------|------|
| HTML | Структура страниц |
| CSS | Адаптивный дизайн (Material Design) |
| JavaScript | Интерактивность, работа с данными |
| localStorage | Хранение состояния корзины |

## Архитектура прототипа

### Структура приложения
- **Одностраничное приложение (SPA)** с разделением на HTML, CSS и JS.
- **Модульная структура:** `index.html` (разметка) + `style.css` (стили) + `main.js` (логика).
- **Ключевые функции:** `showBooks()`, `addToCart()`, `filterBooks()`, `submitOrder()`.

### Компоненты приложения

**index.html (разметка):**
- Шапка сайта (логотип, поиск, корзина со счётчиком)
- Категории (Все, Художественная, Учебная, Детская)
- Каталог книг (отображение карточек товаров)
- Модальные окна (корзина, оформление заказа, успешный заказ)

**main.js (логика):**
- `books []` - Массив книг (55+ позиций)
- `cart []` - Корзина пользователя
- `showBooks()` - Отрисовка каталога
- `addToCart()` - Добавление в корзину
- `filterBooks()` - Фильтрация по категориям
- `updateCartCount()` - Обновление счётчика
- `changeQuantity()` - Изменение количества
- `removeFromCart()` - Удаление из корзины
- `openCart()` - Открытие корзины
- `openCheckout()` - Оформление заказа
- `submitOrder()` - Отправка заказа
- `localStorage` - Хранение данных

**style.css (стили):**
- Адаптивный дизайн (Flexbox/Grid сетки)
- Медиа-запросы для мобильных устройств
- Анимации и переходы
- Принципы Material Design

## Основной функционал

### 1. Каталог книг
- **Отображение каталога книг** с фильтрацией по категориям:
  - Все книги (55+ позиций)
  - Художественная литература (15 книг)
  - Учебная литература (15 книг)
  - Детская литература (25 книг)
- **Поиск книг** по названию или автору в реальном времени

### 2. Корзина покупок
- **Добавление/удаление** книг из корзины
- **Изменение количества** товаров в корзине (+ / -)
- Автоматический пересчёт общей суммы
- Визуальный счётчик в шапке сайта
- **Сохранение состояния** корзины между сессиями через `localStorage`

### 3. Оформление заказа
- **Форма с валидацией полей:**
  - Имя покупателя (обязательное)
  - Телефон (обязательное)
  - Адрес доставки (обязательное)
  - Способ оплаты (выпадающий список)
  - Комментарий к заказу (опционально)
- Отображение итоговой суммы заказа
- **Подтверждение заказа** с выводом деталей в консоль

### 4. Модальные окна
- Корзина с детальным просмотром товаров
- Форма оформления заказа
- Подтверждение успешного заказа
- Плавные анимации открытия/закрытия
